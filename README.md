# 💨 Architects — 실시간 대기질 모니터링 웹 서비스

## ✨ 프로젝트 소개  
**Architects**는 전국의 **실시간 미세먼지 및 대기질 정보를 시각적으로 제공**하는 반응형 웹 서비스입니다.  
사용자는 현재 위치 또는 선택한 지역의 대기질을 한눈에 확인하고, 최근 추이를 그래프로 살펴보며, 농도 단계별 행동 요령도 함께 확인할 수 있습니다.  

---

## 🛠️ 기술 스택  
- **프론트엔드:** React, Vite, JavaScript (ES6+), HTML, CSS  
- **API:** 공공데이터포털 (AirKorea) Open API  
- **추가 기능:** PWA(Progressive Web App) 지원으로 모바일 환경 최적화  
- **배포:** (예: Netlify / Vercel / GitHub Pages 사용 시 여기에 추가 가능)

---

## ⚙️ 주요 기능  

### 🌫 실시간 미세먼지 정보  
- 현재 위치 및 선택한 지역의 **미세먼지(PM10) / 초미세먼지(PM2.5)** 농도(㎍/㎥)와 등급 표시  
- **차트:** 최근 4시간 농도 변화를 선형 그래프 및 막대 그래프로 시각화  
- **지도:** 전국 지역별 미세먼지 등급을 색상으로 구분, *hover* 시 상세 수치 표시  

### 🚨 지역별 경보 현황  
- 각 지역의 **미세먼지 경보 발령 및 해제 정보** 제공  
- 발령 시각, 해제 시각 등 세부 정보 포함  

### 🩺 맞춤형 행동 요령  
- 미세먼지 농도 단계별 **권장 행동(예: 마스크 착용, 실외활동 자제)** 안내  
- 고농도 시 추가 행동 요령 제공  

### 🧩 서비스 소개 페이지  
- 서비스 기획 의도 및 기능 설명  
- 개발자 소개 섹션 포함  

### 📱 사용자 친화적 UI/UX (PWA 지원)  
- 모바일 및 데스크톱 **반응형 디자인**  
- **로고, 햄버거 메뉴, 헤더, 내비게이션, 푸터** 등 통일성 있는 구조  
- 깔끔하고 직관적인 인터페이스

## 🚀 실행 방법  

### 1️⃣ 레포지토리 클론
```bash
git clone https://github.com/itsurfavsong/architects.git
cd architects
```
### 2️⃣ 패키지 설치
```bash
npm install
```
### 3️⃣ 개발 서버 실행  
```bash
npm run dev
```
